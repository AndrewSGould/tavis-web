Bones - qualifying completions

The scoring will work as follows:

  • > 1 GB Install Size = 20% of BCMX value + (XX% of BCMX Value)
  • >= 5 GB Install Size = 40% of BCMX value + (XX% of BCMX Value)
  • >= 10 GB Install Size = 60% of BCMX value + (XX% of BCMX Value)

To gain a higher XX% - we're asking you upstanding members of society to empower the bones that you deliver with dark magic. For each ritual that you manage to place upon your bones an additional 5% will be granted (to a maximum of 25%). This additional total will apply to all games that work for the bonus.

If applicable - the same completion can be used in multiple rituals. It will not require five unique completions.

  • Crimson Curse
    • This ritual will be completed by completing a game with the string “red” appearing anywhere in the title. System tags will not count. (+5%)

  • Dread
    • This ritual will be completed by completing a game that has a start date (in your collection) of 2023 or earlier. (+5%)

  • Mark of the Beast (1 of 3)
    • This ritual will be completed by completing a game that has a “6” appearing in the achievement count (including all DLC). (+5%)

  • Mark of the Beast (2 of 3)
    • This ritual will be completed by completing a game that has a “6” appearing in the release date. US dates on TA will be used. (+5%)

  • Mark of the Beast (3 of 3)
    • This ritual will be completed by completing a game that has a “6” appearing in the larger number in the “completion est inc DLC” column in the game collection. The game has to have enough estimates for a value to show here. If a value does not show, it will not count. TAVIS only sees the larger number. For example 6-8 would count as an 8 (the 6 would not be seen.) (+5%)
